# 功能介绍
在tensorflow框架下，我们构建了一个卷积神经网络用于分辨猫和狗的图片。此卷积神经网络主要包括三个卷积层，一个全连接层。输入格式为图片。
# 训练数据文件结构
```
-----train_data
--------dog.1.jpg
--------dog.2.jpg
--------dog.3.jpg
--------*********
```
